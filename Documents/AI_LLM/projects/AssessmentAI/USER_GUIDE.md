# AssessmentAI - User Guide

## üöÄ Quick Start Guide

### Step 1: Create Your Account
1. Click **"Register"** in the top right corner
2. Enter your email and password
3. Check your email for confirmation link
4. Click the confirmation link to activate your account

### Step 2: Log In
1. Click **"Login"** in the top right corner
2. Enter your email and password
3. You'll be redirected to the homepage

### Step 3: Create Your First Assessment
1. Click **"Manage Questions"** in the navigation bar
2. Click **"Create New Assessment"** button
3. Fill in the assessment details:
   - **Name**: Give your assessment a clear name (e.g., "JavaScript Basics")
   - **Description**: Brief description of what it covers
   - **Icon**: Choose an icon that represents your topic
   - **Color**: Pick a color theme
   - **Visibility**: Choose Public or Private

### Step 4: Upload Questions via Excel
1. After creating the assessment, you'll see the upload area
2. **Download the Excel template** by clicking "Download Template"
3. **Fill in your questions** in the Excel file:
   - Column A: Question text
   - Column B: Option A
   - Column C: Option B
   - Column D: Option C
   - Column E: Option D
   - Column F: Correct answer (A, B, C, or D)
4. **Save your Excel file**
5. **Drag and drop** the file or click "Select Excel file"
6. Click **"Upload Questions"**

### Step 5: Take Your Assessment
1. Go to **"Assessments"** in the navigation
2. Find your assessment and click **"Start Assessment"**
3. Read the instructions and click **"Start Assessment"**
4. Answer questions one by one
5. Use the **"AI Help"** button if you need hints or explanations
6. Click **"Submit Assessment"** when done

### Step 6: View Results
1. After submission, you'll see your results
2. Review your score and detailed feedback
3. Check the **"Dashboard"** to see all your assessment history

1. **Hints**: Strategic guidance without revealing answers

## üìù Excel Template Format

Your Excel file should have these columns:
```
| Question | Option A | Option B | Option C | Option D | Correct Answer |
|----------|----------|----------|----------|----------|----------------|
| What is 2+2? | 3 | 4 | 5 | 6 | B |
```

### Important Notes:
- **Correct Answer** should be A, B, C, or D (not the actual text)
- **Maximum file size**: 10MB
- **Supported formats**: .xlsx, .xls
- **Minimum questions**: 1
- **Maximum questions**: No limit

## üîß Troubleshooting

### Can't see my assessments?
- Make sure you're logged in
- Check if you created the assessment under the correct account
- Try refreshing the page

### Excel upload not working?
- Check file format (.xlsx or .xls)
- Ensure file size is under 10MB
- Verify all required columns are present
- Make sure "Correct Answer" column uses A, B, C, or D

### AI Help not working?
- Make sure you're in an active assessment
- Check your internet connection
- Try refreshing the page

### Assessment shows "No questions available"?
- Make sure you uploaded questions successfully
- Check if the Excel file had the correct format
- Try uploading the questions again

## üí° Tips for Better Assessments

### Writing Good Questions
- Keep questions clear and concise
- Avoid ambiguous wording
- Make sure only one answer is clearly correct
- Include a mix of difficulty levels

### Using AI Help Effectively
- Try to answer questions yourself first
- Use hints to guide your thinking process
- Use explanations to understand concepts deeply
- Don't rely solely on AI - use it as a learning tool

### Managing Multiple Assessments
- Use descriptive names for easy identification
- Organize by topic or difficulty level
- Use colors to categorize different subjects
- Regularly review and update your questions

## üéØ Best Practices

### For Educators
